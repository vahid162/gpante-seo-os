# WooCommerce URL Policy

Classify product URLs, categories, product tags when present, product attributes, variations, search URLs, filters, layered navigation, pagination, cart, checkout, My Account, Store API, REST, and `wc-ajax` separately. Do not apply one indexation rule to all WooCommerce URLs.

For each URL class record search purpose, inventory relationship, canonical state, sitemap eligibility, schema expectations, internal links, customer-flow risk, and validation requirements.
