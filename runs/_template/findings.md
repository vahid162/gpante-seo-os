# Run Findings

This file is the canonical Finding register for this Run. Findings are not owned by `audits/` and this repository does not maintain a global Finding registry in this version.

Finding statuses must use `../../workflows/status-definitions.md`. Finding IDs use the `FND-001` format and are unique within this Run. `FND-001` is an example format only, not an assigned identifier.
