# Run Evidence

Evidence records and safe evidence references for this Run belong here.

Evidence metadata follows `../../../policies/evidence-policy.md`. Raw, reference-only, prohibited, unsanitized, secret, credential, customer, database, backup, log, or access-granting material must not be stored in Git.

Evidence IDs use the `EV-001` format and are unique within this Run. `EV-001` is an example format only, not an assigned identifier.
