# Task Backlog

This file is the current cross-run Task registry for discoverability and canonical status, and is planned to become a derived/generated backlog in a future phase. Individual Task records are the record-first source of truth for each Task. This is a registry/status index only: it does not own approval evidence, implementation history, detailed Validation, rollback details, or monitoring notes, and it does not replace the individual canonical Task record. Task IDs use the `TSK-YYYY-NNN` format. `TSK-2026-001` is an example format only, not an assigned identifier.
