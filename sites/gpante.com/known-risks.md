# gpante.com Known Risk Model

This document lists risk categories for future review. It does not claim these risks are currently active unless a dated Run records supporting evidence.

## Known Risks

- Dynamic WooCommerce endpoint performance.
- Faceted-navigation crawl waste.
- Cache safety for session-dependent pages and endpoints.
- Redirect chains.
- Invalid historical URLs.
- Cron and Action Scheduler interactions.
- Resource constraints affecting crawl and user experience.
- Duplicate or thin product/category content.

## Open Questions

- Which risks are active in the current production environment.
- Which URL classes have measurable crawl demand or search demand.
- Which cache states are observed for anonymous, authenticated, cart, checkout, REST, and AJAX responses.
