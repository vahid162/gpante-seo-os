# gpante.com URL Classes

Use the canonical model in `../../technical-seo/url-classification.md`.

## Site-Relevant Classes

- Homepage.
- WooCommerce product pages.
- Product categories.
- Product tags and attributes if present.
- Faceted/filter and layered navigation URLs.
- Blog/article content.
- Forum/community URLs where wpForo is present.
- Cart, checkout, My Account, login, REST, Store API, and WooCommerce AJAX endpoints.
- Media, feeds, pagination, redirected URLs, and invalid historical URLs.

No current URL inventory is recorded here; concrete inventories belong in a Run.
