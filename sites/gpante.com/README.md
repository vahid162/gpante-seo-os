# gpante.com Site Knowledge Contract

## Purpose

`sites/gpante.com/` owns sanitized, reviewed, site-specific operational knowledge for gpante.com.

## What belongs here

- Confirmed facts, historical facts, known risks, open questions, and inferences clearly labeled.
- Repository-safe summaries of platform, ecommerce, SEO, and infrastructure context.
- Version snapshots only when dated and supported by evidence.

## What does not belong here

- Credentials, tokens, cookies, usernames, customer data, order data, raw logs, database exports, private URLs, or unsanitized production evidence.
- Executed Findings, Run validation results, canonical Decisions, or canonical Task status.
