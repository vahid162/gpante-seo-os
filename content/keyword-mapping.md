# Keyword Mapping Model

A keyword map is a planning model, not ranking evidence by itself.

## Required Fields

| Field | Requirement |
| --- | --- |
| Primary query | Main query supported by evidence; no fabricated volume. |
| Secondary queries | Supporting terms, variants, and modifiers. |
| Search intent | Transactional product, commercial investigation, category, informational, comparison, brand, or support/troubleshooting. |
| Target URL | Existing or proposed URL; classify URL type. |
| URL type | Product, category, article, forum/community, support, or other documented class. |
| Funnel stage | Awareness, consideration, purchase, support, retention. |
| Cannibalization risk | Existing competing URLs and evidence. |
| Current ranking evidence | Source, date, scope, and limitation. |
| Business priority | Rationale linked to products/categories or strategy. |
| Content status | Existing, needs refresh, gap, consolidate, no action. |
| Internal-link requirements | Required sources and destinations. |
